# [How to use](https://github.com/enigmapr0ject/SCR-SGPlus/blob/master/README.md)

### **Installation**

## macOS
1. Download and install the latest Python3 installer from [Python Downloads Page](https://www.python.org/downloads/)

<img width="1440" alt="" src="https://user-images.githubusercontent.com/77569325/212567542-fdb8a1b8-8be8-4c29-b744-471a752c6b18.png">
<img width="732" alt="" src="https://user-images.githubusercontent.com/77569325/212567592-97e7e3e4-02e4-4530-9e41-f7dea03a28a0.png">

2. Download the SG+0.4 for Mac [Download Link](https://github.com/BananaCakeComputer/SCR-SGPlus-macOS/archive/refs/heads/master.zip).

3. Extract the folder to your desired location. Unzip the zip file.

<img width="408" alt="" src="https://user-images.githubusercontent.com/77569325/212567656-2ced5850-d2dc-4871-816e-aa7c11b47948.png">

4. Open the Terminal`/System/Applications/Utilities/Terminal.app`. In the Terminal, type `pip3 install -r `. Frag the `requirements.txt` to the Terminal window in the folder where the script and images are stored, then press Enter.

![](https://user-images.githubusercontent.com/77569325/212568107-59acb086-8868-4f5e-99a1-80f87069f933.gif)

You should get something like this:

<img width="682" alt="" src="https://user-images.githubusercontent.com/77569325/212568157-36b22aee-9b9b-4278-9c84-6537f41e510a.png">

5. After the installation has completed, run the `overlay.py` file to open the script. Press `fn+F5` to start the script.

<img width="811" alt="" src="https://user-images.githubusercontent.com/77569325/212568450-1fac4325-92b7-4e09-9d74-186b611d6195.png">

6. If you run the script and Start the SG+ in the menu, you may see something like this:

<img width="573" alt="" src="https://user-images.githubusercontent.com/77569325/212450696-9f8e4953-6301-4249-9ccd-9fba02d710eb.png">

Which means you have to able the screen record for Python

Check out `Settings>Security & Privacy>Screen Record` and allow "Python"

<img width="827" alt="" src="https://user-images.githubusercontent.com/77569325/212450802-4da5452b-9837-4c7f-a595-e68f288c4802.png">
